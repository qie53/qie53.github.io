springBoot整合mybatis

  1. 创建mapper.UserMapper.interface

    1. package com.edu.penguin.mapper;

import com.edu.penguin.pojo.User;
import org.apache.ibatis.annotations.Mapper;
import org.springframework.stereotype.Repository;

import java.util.List;

@Mapper
@Repository
public interface UserMapper {
    List<User> selectAllList();

    User select(Integer id);

    int deleteUser(Integer id);

    int addUser(User user);

    int updateUser(User user);
}

<!--￼4-->

  2. 在yml中配置配置位置

    1. mybatis:
  type-aliases-package: com.edu.penguin.mapper.UserMapper
  mapper-locations: classpath:mybatis/mapper/*.xml

注解

  • @Component：组件放在类上，说明这个类被spring管理了，就是bean，用小写类访问（下面这三个注解和这个注解一样，都表示装配Bean，只是为了表示）写在pojo层

    • @Repository：dao层
    • @Service：service层
    • @Controller：controller层

  • @Value：用法如下直接进行注入赋值

  • @Autowired：自动装配，通过类型，名字

    • 如果不能唯一自动装配，则需要通过@Qualifier(value = “name”)

  • @ConfigurationProperties：如果你使用properties或yml进行注入就需要在想要注入的类上面写上 @ConfigurationProperties(prefix=”cat”) prefix是指定配置文件中的对应的注入

  • @PropertySource：加载指定的配置文件 用法

    • @PropertySource(value = “classpath:文件名会从resource出发”)

  • @Validated：JSR303校验 进行各种校验要导入包

    • <dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-validation</artifactId>
</dependency>

    • @Email()

    • @NotNull()：注入不能为空

    • @Max(value = 100,message = “最大不超过100，提示消息”)：最大不超过100

    • @Min(0)：最小不低于0

  • @responseBody：视图解析器不生效

  • @RestController：不解析视图的controller
